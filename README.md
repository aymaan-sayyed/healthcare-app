# 🏥 Jarurat Care - Healthcare Support App

## 🔗 Live Demo
https://healthcare-app-tau-tawny.vercel.app/

---

## 📌 Overview
Jarurat Care is a simple healthcare support web app designed for NGOs to connect patients and volunteers efficiently. Users can submit requests and get instant guidance through an AI assistant.

---

## ⚙️ Tech Stack
- React (Vite)
- JavaScript
- HTML & CSS
- Vercel (Deployment)

---

## 🤖 AI / Automation Idea
The app includes a basic AI chatbot that provides instant responses to common health queries such as fever, headache, COVID, etc.  
This helps reduce manual workload for NGOs by handling repetitive queries automatically.

---

## 🧩 NGO Use Case
- Patients can submit their health issues or requests for help  
- Volunteers can register to assist  
- AI assistant provides quick initial guidance  
- NGOs can manage requests efficiently
